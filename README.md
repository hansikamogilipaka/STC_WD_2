# STC_WD_2# ⏱ Stopwatch Web Application

An interactive and user-friendly stopwatch web application developed using HTML, CSS, and JavaScript. This project allows users to accurately measure time intervals with features like start, pause, reset, and lap tracking.

---

## 🚀 Features

- ▶️ Start the stopwatch
- ⏸ Pause the stopwatch
- 🔄 Reset the timer
- 🏁 Record lap times
- ⏱ Millisecond precision
- 🎨 Modern and responsive UI design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
